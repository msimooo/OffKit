01
What makes this different
🔒
Privacy by design
All your data stays in your browser's local storage. Nothing is transmitted to any server. No accounts, no tracking, no analytics. Your data is yours — full stop.
⚡
Instant & offline
No loading screens or login walls. Works completely offline once the page is open.
📦
One portable file
The entire suite — all seven apps — lives in a single HTML file you can save anywhere and share freely.
📱
Mobile-first
Designed for touch with fluid layouts that feel native on phones and tablets.
🎨
Considered design
Every spacing, colour, and animation decision is intentional — calm and focused, never noisy.
🔧
Zero dependencies
Pure HTML, CSS and JS. No frameworks, no build tools, no package managers required.
02
Built with the web platform

Every app is written in plain HTML, CSS, and JavaScript. Data persistence uses the browser's built-in localStorage — so your data stays local and always belongs to you.
